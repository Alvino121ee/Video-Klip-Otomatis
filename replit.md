# YouClip — AI Auto Clipper

Paste a YouTube link and AI automatically picks the best moments, cuts them vertically, and adds subtitles — ready for TikTok, Reels, and Shorts.

## Run & Operate

- **Clipper workflow** (port 5173) — React/Vite frontend (`pnpm --filter @workspace/clipper run dev`)
- **API Server workflow** (port 8080) — Express 5 backend (`PORT=8080 pnpm --filter @workspace/api-server run dev`)
- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas from the OpenAPI spec
- `pnpm --filter @workspace/db run push` — push DB schema changes (dev only)
- Required env: `DATABASE_URL` (auto-provisioned), `DEEPSEEK_API_KEY` (set as secret)

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- API codegen: Orval (from OpenAPI spec)
- Build: esbuild (CJS bundle)
